
## Breakpoint Variables

All breakpoints are defined in `_sass/_variables.scss` and should be used consistently across all SCSS files.

| Variable Name | Value | Description | Usage |
|---------------|-------|-------------|-------|
| `$bp-mobile` | 480px | Small mobile devices | Navigation collapse, small screen layouts |
| `$bp-mobile-small` | 576px | Mobile devices | Single column layouts |
| `$bp-tablet-small` | 600px | Small tablets | Gallery grid adjustments |
| `$bp-tablet` | 860px | Tablets and large mobile | Most common responsive breakpoint |
| `$bp-desktop-nav` | 940px | Desktop navigation threshold | Header navigation layout changes |
| `$bp-desktop` | 992px | Standard desktop | Grid layout changes |
| `$bp-desktop-large` | 1000px | Large desktop | Container width adjustments |
| `$bp-desktop-xl` | 1400px | Extra large screens | Logo positioning, container sizing |
| `$bp-desktop-ultra` | 1800px | Ultra-wide screens | Maximum container width |

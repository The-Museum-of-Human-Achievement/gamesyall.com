L1TTL3 PAWS is a physics based arcade game where you play as a cat gliding over rolling hills while collecting flowers. Spend flowers to unlock 13 different playable cats, each with its own special traits. Completing all 13 islands unlocks a remix mode that is procedurally generated to be different every run. Originally created for the JS13K competition using, the entire game fits in a 13 kilobyte zip file, and the full source code is available on GitHub.
